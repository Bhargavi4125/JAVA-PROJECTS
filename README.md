# Simple ATM System (Java)

## Overview
This is a console-based ATM simulation system developed using Java. The project allows users to perform basic banking operations such as checking balance, depositing money, withdrawing money, and viewing transaction history after PIN authentication.

---

## Features
- PIN-based login authentication  
- Check account balance  
- Deposit money  
- Withdraw money  
- Transaction history tracking  
- Input validation using exception handling  
- Basic error handling for incorrect PIN and insufficient balance  

---

## Technologies Used
- Java  
- java.util (Scanner, ArrayList)  


## How to Run

### Step 1: Compile the program
```bash
javac SimpleATM.java
```

### Step 2: Run the program
```bash
java SimpleATM
```
